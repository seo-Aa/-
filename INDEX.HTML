HTML (Hypertext Markup Language) is not a programming language, but a markup language that allows the browser to see how the web pages we view are structured. It can be programmed complexly and simply for developers. HTML consists of elements, which wrap and mark up different parts of each content to represent and execute in an appropriate way. Tags are responsible for creating hyperlinks that move you to other pages on the web, or for highlighting words
